## Installation
Follow these steps to install UDE:
1. **Clone the repository**:
   ```bash
   git clone https://github.com/geeberry07/DesignCompiler.git
   cd DesignCompiler
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
